# ITrip Project


## Installation

```sh

npm init -y

npm i express                   // Express
npm i express-validator         // Validator fields
npm i bcryptjs                  // Encriptar password

npm i dotenv                    // DotEnv
npm i cors                      // Cors

```


## Run Server

```sh

node app.js

nodemon app.js

```



## Sequelize - ORM

```sh

npm install sequelize mysql2

```
