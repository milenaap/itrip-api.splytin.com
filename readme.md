# ITrip Project


## Installation

```sh

npm init -y

npm i express                           // Express
npm i express-validator                 // Validator fields
npm i bcryptjs                          // Encriptar password

npm i dotenv                            // DotEnv
npm i cors                              // Cors


npm install sequelize mysql2            // ORM Sequelize


npm install module-alias                // Alias @ src


```


## Run Server

```sh

node app.js

nodemon app.js

```
